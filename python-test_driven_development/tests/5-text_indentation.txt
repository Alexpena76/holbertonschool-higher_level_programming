The ``5-text_indentation`` module
======================

Using ``text_indentation``
-------------------

Importing the function from the module:
    >>> text_indentation = __import__('5-text_indentation').text_indentation

Basic text with special characters:
    >>> text_indentation("Hello world.")
    Hello world.
    <BLANKLINE>

    >>> text_indentation("How are you?")
    How are you?
    <BLANKLINE>

    >>> text_indentation("Here is the text: important info.")
    Here is the text:
    <BLANKLINE>
    important info.
    <BLANKLINE>

Text with spaces after special characters:
    >>> text_indentation("Test.   Next sentence.")
    Test.
    <BLANKLINE>
    Next sentence.
    <BLANKLINE>

    >>> text_indentation("Question?    Another question?")  
    Question?
    <BLANKLINE>
    Another question?
    <BLANKLINE>

Multiple special characters:
    >>> text_indentation("First. Second: Third?")
    First.
    <BLANKLINE>
    Second:
    <BLANKLINE>
    Third?
    <BLANKLINE>

Text without special characters:
    >>> text_indentation("No special characters here")
    No special characters here

Empty string:
    >>> text_indentation("")

Error cases - Non-string input:
    >>> text_indentation(None)
    Traceback (most recent call last):
        ...
    TypeError: text must be a string

    >>> text_indentation(123)
    Traceback (most recent call last):
        ...
    TypeError: text must be a string
